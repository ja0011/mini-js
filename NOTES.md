# mini-js — notes index

Small projects live in separate folders. Open the folder you are working on (*File → Open Folder…*) so paths and Live Server match that project.

Each **`pj*/NOTES.md`** explains that mini-project’s HTML/CSS/JS. Topics are split with **`---`** breaks, **`###` headings**, and short bullets so nothing is one giant wall of text. JavaScript ideas are grouped under small **“Vocabulary”** headings (**`const`**, **`function`**, **`let`**, etc.).

| Folder | What it is | Notes |
|--------|------------|--------|
| **`pj1/`** | Color flipper (`main.html`, `index.js`, `style.css`) | `pj1/NOTES.md` |
| **`pj2/`** | Palindrome checker (`main.html`, `index.js`; `style.css` linked but not in repo yet) | `pj2/NOTES.md` |
| **`pj3/`** | Random quote generator (`index.html`, `index.js`) | `pj3/NOTES.md` |
| **`pj4/`** | Stop watch (`index.html`, `index.js`) | `pj4/NOTES.md` |
| **`pj5/`** | TODO list with `localStorage` (`index.html`, `index.js`) | `pj5/NOTES.md` |
